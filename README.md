# Teacher Toolkit

A working app for teachers built with Claude.

## Files

- `teachertoolkit.html` - Main application file

## How to Use

1. Open `teachertoolkit.html` in your web browser
2. Or deploy using GitHub Pages (see instructions below)

## Deployment with GitHub Pages

To make this app available on the web:

1. Go to repository Settings → Pages
2. Select "Deploy from a branch"
3. Choose the branch (main) and folder (root)
4. Your app will be available at `https://Senseitoolkit.github.io/test-repo/teachertoolkit.html`
